# BEDLAM360 Legacy Versions

This folder mirrors the older v0/v1 dataset generators that are not needed for the frozen v3 render path.

## Contents
- `bedlam360_generate_v0_dataset.py`: legacy v0 dataset pipeline entrypoint. Keep only for historical reproduction.
- `bedlam360_generate_v1_dataset.py`: legacy v1 dataset pipeline entrypoint. Keep only for historical reproduction.

These scripts still depend on the live `bedlam360_pipeline/` helpers in the source tree.
